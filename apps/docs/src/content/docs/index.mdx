## Project Overview

**Rotunde Music** is a modern monorepo project built with Turborepo, managing multiple applications and shared packages for a music-focused content management system.

### Architecture

This project uses a **monorepo architecture** powered by:
- **Turborepo** - High-performance build system
- **pnpm** - Fast, disk space efficient package manager
- **TypeScript** - Type-safe development across all packages

## Quick Start

### Prerequisites

- **Node.js** (v18 or higher)
- **pnpm** (v8.15.6) - Install with `npm install -g pnpm@8.15.6`

### Development

Run all applications in development mode:

```bash
pnpm dev
```

Run specific tasks:

```bash
# Build all packages
pnpm build

# Lint all packages
pnpm lint

# Format code
pnpm format
```
